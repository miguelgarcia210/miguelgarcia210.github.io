Eligible Training Provider System

    The Eligible Training Provider System, also known as E.T.P.S., is a full stack application developed in collaboration with the Texas Workforce Commission and Codeup.
    ETPS was developed to improve the processes between Texas Workforce Commission Program Specialists and Certified Eligible Training Providers.
    These processes include, but are not limited to, training provider program updates, automatic notifications importing and exporting of provider data, as well as secure and direct communication for Texas Workforce Commission program specialists and it's many impacted individual providers. ETPS is the developmental product of a more efficient system maintaining provider security and privacy.
    ETPS utilizes the Model, View, Controller (MVC) design pattern, MySQL as it's RDBMS, Apache POI API towards all Microsoft Excel data manipulation provided either imported and exported.

Movie Project

    "Movie Project" is a web application built using JavaScript and jQuery. It's entire concept is based towards a movie logging application. By utilizing JavaScript, users are able to Create, Read, Update, and Delete movie data. This applications UI was created using a CSS framework known as Bootstrap along with the implementation of custom CSS.
    Utilizing jQuery, Ajax requests were performed to send and retrieve data without refreshing the current page. The current web page would then render generated HTML based on JSON responses. With editing movies, users are prompted with a pre-populated form with the selected movie's data to be altered and updated.
